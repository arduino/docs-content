<FeatureDescription>

The Modulino Hub is an I2C multiplexer featuring eight independent channels, allowing you to connect multiple sensors with identical addresses. Built around the TCA9548ARGER switch, it provides a clean solution for organizing complex I2C networks and expanding your project's sensor capabilities.

</FeatureDescription>

<FeatureList>

<Feature title="8 Independent Channels" image="connection">

Connect up to eight I2C devices per Hub, each on its own isolated channel. Perfect for using multiple identical sensors in a single project.

</Feature>

<Feature title="Address Conflict Resolution" image="configurability">

Eliminate I2C address conflicts by isolating devices on separate channels. No more worrying about sensors sharing the same address.

</Feature>

<Feature title="Scalable Design" image="communication">

Chain up to eight Hubs on the same I2C bus for a total of 64 independent channels. Configure addresses using onboard solder jumpers.

</Feature>

<Feature title="Hardware Address Configuration" image="settings">

Configure the Hub's I2C address using three solder jumpers on the back. Choose from eight addresses (0x70-0x77) to run multiple Hubs simultaneously.

</Feature>

</FeatureList>
