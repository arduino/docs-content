<FeatureDescription>
The Modulino® Light module features an advanced optical sensor providing ambient light, RGB colour detection, and infrared sensing capabilities. It is designed to be used with any compatible board with Qwiic, allowing you to build smart lighting, colour recognition, and optical sensing projects without making complicated connections. The module also includes example projects for you to learn important programming concepts and get inspired.
</FeatureDescription>


<FeatureList>
<Feature title="RGB Colour Detection" image="color-sensor">
This module provides high-precision colour detection with separate red, green, and blue channels, plus ambient light and infrared sensing for comprehensive optical measurements.
</Feature>
<Feature title="Qwiic Connectors" image="connection">
The Modulino® Light module connects to your UNO R4 WiFi with Qwiic cables, letting you focus on learning programming without building complex circuits.
</Feature>

</FeatureList>
