<FeatureDescription>
The Modulino® LED Matrix module features an 8×12 LED matrix display (96 LEDs total) for showing text, graphics, and animations. It is designed to be used with any compatible board with Qwiic, allowing you to build visual displays and interactive projects without making complicated connections. The module also includes example projects for you to learn important programming concepts and get inspired.
</FeatureDescription>


<FeatureList>
<Feature title="96 Programmable LEDs" image="display">
This module provides an 8×12 LED matrix with the same functionality as the Arduino® UNO R4 WiFi built-in matrix, ensuring code compatibility for scrolling text, graphics, and animations.
</Feature>
<Feature title="Qwiic Connectors" image="connection">
The Modulino® LED Matrix module connects to your UNO R4 WiFi with Qwiic cables, letting you focus on learning programming without building complex circuits.
</Feature>

</FeatureList>
