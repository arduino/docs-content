<FeatureDescription>
The Modulino® Latch Relay module features a bistable latching relay that maintains its state without continuous power, capable of switching high-current DC loads up to 30V. It is designed to be used with any compatible board with Qwiic, allowing you to build automation and power control projects without making complicated connections. The module also includes example projects for you to learn important programming concepts and get inspired.
</FeatureDescription>


<FeatureList>
<Feature title="Bistable Operation" image="power">
This module uses a latching relay that maintains its state even when power is removed, ideal for energy-efficient automation and safety-critical applications.
</Feature>
<Feature title="Qwiic Connectors" image="connection">
The Modulino® Latch Relay module connects to your UNO R4 WiFi with Qwiic cables, letting you focus on learning programming without building complex circuits.
</Feature>
