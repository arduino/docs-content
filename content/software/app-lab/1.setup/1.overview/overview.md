Arduino App Lab is the integrated development environment (IDE) specifically designed for the Arduino UNO Q. It allows you to develop, deploy, and manage hybrid applications that combine the power of a Linux microprocessor with the real-time reliability of a microcontroller running Arduino/C++.

## Choose Your Setup Mode

You can run Arduino App Lab in two primary ways depending on your project needs and available hardware.

- **Connected Mode**. Install Arduino App Lab on your host computer and connect to your board via USB.
- **Single-Board Computer (SBC)**. Use the Linux desktop environment and run Arduino App Lab directly on the board.

If you're unsure, Connected Mode is a great way to get started. You can switch between using different modes at any time.
