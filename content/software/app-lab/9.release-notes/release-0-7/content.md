## Overview

This page contains all release notes for Arduino App Lab. To access the software, go [here](https://www.arduino.cc/en/software/#app-lab-section).

<Alert type="info">Note: You need to have an UNO Q ([2GB](https://store.arduino.cc/products/uno-q) or [4GB](https://store.arduino.cc/products/uno-q-4gb)) to use Arduino App Lab.</Alert>

## Release 0.7.0 [2026.04.29]

### TL;DR

#### What's New

* You can now create your own custom Bricks
* The console panel moved to the editor tab
* `Serial.print()` and `Monitor.print()` works interchangeably

#### What's Fixed

* Improved reliability of Arduino-router communication
* Improved board update process
* Fixed bug while configuring board in SBC mode

### Full Version

**Custom Bricks**

You can now create your own Bricks to extend App Lab with custom behaviors and integrations.

**Smarter startup experience**

App Lab now remembers your last used board and last opened app, so you can pick up exactly where you left off.

#### What’s New

* Improved file management in the editor
* Updated Edge Impulse Model Runner images
* Console integrated into the editor
* Fully integrated console panel (logs + serial monitor) into the editor

#### Improvements & Fixes

* More reliable Arduino-router communication
* Faster, safer board update process
* Better editor experience
* SBC mode configuration fix (ongoing hardening)
* Sound Generator Brick (enhanced)
